# vue-cli-plugin-easy-axios

> 基于easier-axios封装的Vuecli插件

> version:  0.0.10

> lastDate: 2020/3/10

> Author:  Sumer Liu

## Features
1. 以axios作为基础库，easier-axios作为api管理工具对axios请求做模块化管理。
2. 已实现axiso配置，拦截器等模板

## Installatiom
```
 vue add easy-axios

```

## 关于easier-axios
关于easier-axios中模块注册以及全局注册API异步到[easier-axios](https://github.com/sumerliu/esaier-axios)
